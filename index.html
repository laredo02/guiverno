Mis Proyectos

1. Sistema de Gestión de Inventario

Desarrollé un sistema de gestión de inventario que utiliza modelos de predicción basados en series temporales. Una de las fórmulas clave que implementamos fue la suavización exponencial: \[S_t = \alpha \cdot X_t + (1 - \alpha) \cdot S_{t-1}\] Donde: \(S_t\) es el valor suavizado para el período t \(X_t\) es el valor observado en el período t \(\alpha\) es el factor de suavización (0 < \(\alpha\) < 1)

2. Aplicación Web de Optimización de Rutas

Creé una aplicación web que utiliza el algoritmo de Dijkstra para encontrar la ruta más corta entre dos puntos. La fórmula básica para actualizar las distancias es: \[d[v] = \min(d[v], d[u] + w(u,v))\] Donde: \(d[v]\) es la distancia mínima al vértice v \(d[u]\) es la distancia mínima al vértice u \(w(u,v)\) es el peso de la arista entre u y v

3. Modelo de Aprendizaje Automático para Predicción de Precios

Implementé un modelo de regresión lineal para predecir precios. La ecuación del modelo es: \[y = \beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n + \epsilon\] Donde: \(y\) es la variable dependiente (precio) \(x_i\) son las variables independientes \(\beta_i\) son los coeficientes \(\epsilon\) es el término de error
