¿Qué es Python?

Python es un lenguaje de programación muy popular que se usa en muchos ámbitos distintos: desde el desarrollo web y la inteligencia artificial, hasta la ciencia de datos, los videojuegos o la robótica. Su gran ventaja es que la sintaxis es clara y fácil de leer, lo que lo hace ideal para aprender a programar.

A continuación vas a ver algunos ejemplos sencillos de código en Python. No te preocupes si al principio no entiendes todo lo que aparece: es completamente normal. Lo importante ahora es ir echando un vistazo, familiarizarse con la forma que tiene el código, y poco a poco iremos explicando cómo funciona cada cosa durante el curso.

Primeros Pasos

Empezamos con lo clásico: el famoso Hello World. Es la primera línea de código que escribe casi todo el mundo al aprender un lenguaje. 

print("Hello, world!")

Vale, imprimir un texto está bien, pero Python sirve para mucho más. Podemos usarlo para calcular la media de varias asignaturas: 

nota1 = 7.5
	nota2 = 8.0
	nota3 = 6.5

	media = (nota1 + nota2 + nota3) / 3
	print("Tu media es:", media)

En el ejemplo anterior calculamos la media de tres notas usando variables sueltas. Pero ¿qué pasa si en vez de 3 tienes 10 o 20 notas? Sería un lío crear una variable para cada una. Para eso existen las listas, que permiten guardar varios valores en una sola estructura. Después podemos recorrer la lista con un for y procesar todos los elementos de manera automática. 

notas = [7.5, 8.0, 6.5]

	suma = 0
	for nota in notas:
	suma += nota

	media = suma / len(notas)
	print("Tu media es:", media)

Y ahora, el reto: vamos a hacer un pequeño juego. El ordenador piensa un número y tú tienes que adivinarlo. Es un clásico, pero engancha mucho cuando lo pruebas: 

import random

	numero_secreto = random.randint(1, 10)
	intento = int(input("Adivina un número entre 1 y 10: "))

	if intento == numero_secreto:
	print("🎉 ¡Has acertado!")
	else:
	print("😢 No era ese, el número correcto era", numero_secreto)

Fuerza de Gravitación Universal

Hemos reservado este ejemplo para los más pros, aquellos a quienes lo anterior les haya resultado muy fácil.

La gravedad mantiene los planetas en órbita y nos mantiene pegados a la Tierra. Con Python podemos calcular la fuerza de atracción entre dos cuerpos usando la fórmula de Newton: 

G = 6.674 * (10**-11)  # Constante gravitatoria en N·m²/kg²
	
	m_tierra = 5.972 * (10**24)  # kg
	m_luna = 7.348 * (10**22)    # kg
	distancia = 384400000        # m (384.400 km)
	
	F = G * (m_tierra * m_luna) / (distancia**2)
	print("La fuerza gravitatoria entre la Tierra y la Luna es:", F, "N")

Puedes cambiar los valores de masa y distancia para calcular la fuerza gravitatoria entre otros cuerpos, como entre la Tierra y un satélite artificial.
