Introducción a Python

Python es un lenguaje fácil de aprender y muy versátil. Vamos a ver algunos conceptos básicos:

1. Hello World

print("¡Hola, Mundo!")

2. Variables

nombre = "Ana"
edad = 12
print("Hola, " + nombre + ", tienes " + str(edad) + " años")

3. If Statements

edad = 18
if edad >= 18:
    print("Eres mayor de edad")
else:
    print("Eres menor de edad")
